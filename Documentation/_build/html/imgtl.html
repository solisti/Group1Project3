imgtl module¶

imgtl.ocr_core(filename, language)¶

Handles the core OCR processing of the input image.

Parameters

  • filename (string) – the name of the input image file

  • language (string) – the choice of input file language

imgtl.translate_text(image_path, imagename, language)¶

Takes lines of text extracted from txt file and translates them, outputs translated version into a txt file called “translated.txt” and an image file with the translated text called Capture.JPG.

Parameters

  • image_path (string) – the path of the input file obtained from the user

  • imagename (string) – name of the input image file

  • language (string) – the choice of input file language

imgtl.writefile(image_path, strstr)¶

Writes text extracted from image into a txt file named “script.txt” which is saved in the same path as the input image file.

Parameters

  • image_path (string) – the path of the input file obtained from the user

  • strstr (string) – string of words that have been generated by OCR
